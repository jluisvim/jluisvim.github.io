When writing a text 📝...

I'm a faithful supporter of Vim and LaTeX (maths symbols : Detexify , presentations: Beamer). If you would like to be part of the Vim world, these tutorials can be a great help, in my opinion they are very educational: ThePrimeagen, LukeSmith.

Some additional tools for spellchecking: DeepL, Merrian-Webster and Collins dictionaries.

Some links here for calls for papers: CORE and Conference Ranks (conferences and journal ranking), SJR (journal ranking), WikiCFP (for conferences, journals, symposia...), and jackietseng.github site.

When coding 🚧...

I strongly use swi-prolog for theorem prover, database query, and more general as declarative programming, likewise, I use other programming languages in robotics (Python, C/C++), in web page design (HTML/CSS), in embedded programming (Python, C/C++, assembly) and others (bash).
I usually work with operating systems based on UNIX, so Macintosh and Linux are my preferred workspaces.
